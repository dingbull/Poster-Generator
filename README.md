# RewindPPF — Facebook Marketplace Poster Generator

A free, offline web app for creating professional automotive service ads for Facebook Marketplace.

**Live Demo:** [your-username.github.io/rewindppf](https://your-username.github.io/rewindppf)

## Features

### 🖼️ Poster Generator
- Upload any car/work photo as background
- 8 auto-detailing service presets (PPF, Ceramic, Vinyl Wrap, Chrome Delete, etc.)
- 20 colour palettes × 5 colours each
- 20 overlay effects (multi-select, combinable)
- 10 frame/deco styles with T/R/B/L side selection
- 12 text styles (Normal, Glow, Neon, Emboss, Shadow, etc.)
- 8 CTA button styles
- Individual colour pickers for title, tagline, bullets, bold phrases, CTA
- Text position left/right
- Download as 1200×900px PNG — Facebook Marketplace ready

### ✂️ Background Remover
- **100% free, works offline, no AI needed**
- **Colour Tolerance** — click to sample background colour, adjust tolerance
- **Edge Cut** — automatic edge detection using Sobel algorithm
- Gaussian feather for smooth edges
- One-click "Use in Poster" to transfer result directly

## Deploy to GitHub Pages (free)

1. Create a free account at [github.com](https://github.com)
2. Create new repository named `rewindppf` (Public)
3. Upload `index.html` and `README.md`
4. Go to **Settings → Pages → Source: Deploy from branch → main → / (root)**
5. Your app goes live at `yourusername.github.io/rewindppf`

## Privacy

All processing happens **locally in your browser**. No data, photos or text are ever sent to any server.
